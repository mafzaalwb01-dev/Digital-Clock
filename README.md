 # Digital Clock

## About Project

This is a Digital Clock project built using HTML, CSS, and JavaScript.

The clock displays the current time in real-time and updates every second automatically.

## Features

* Live Digital Clock
* Current Date
* Greeting Message
* Responsive Design
* Modern User Interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

The project uses JavaScript Date Object to get the current time and updates the clock every second.
